# Titanic Survival Analysis
A mini data science project analyzing survival patterns on the Titanic dataset.

## Project Goals
- Explore dataset features (age, gender, class, fare).
- Visualize survival rates.
- Build a simple classification model.

## Tech Stack
- Python (Pandas, Matplotlib, Scikit-learn)
- Jupyter Notebook

## Results
- Women had higher survival rates.
- Higher-class passengers had better chances.
- Logistic Regression achieved ~80% accuracy.
